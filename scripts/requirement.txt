create a node application by running the ff. commands

node init

#edit the entry point to server.js

then install the following dependencies:

- "body-parser": 1.19.0"
command: npm install body-parser --save

- "express": "4.17.1"
command: npm install express --save

- "mysql": "2.18.1"
command : npm install mysql --save

to run the node application download nodemon:

- "nodemon": "2.0.4"
command: npm install --save-dev nodemon
